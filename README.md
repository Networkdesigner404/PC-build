# PC-build
Executed a full gaming PC platform migration, replacing an ASUS Prime X470-Pro motherboard with an ROG Strix Z390-E, upgrading the processor from an AMD Ryzen 2900X to an Intel Core i9-9700K, and swapping a GTX 1060 6GB for an EVGA RTX 3080 Ti. This PC upgrade required full OS and driver reconfiguration across a cross-platform CPU change.
